<h2>Installation instructions:</h2>

1. Install the JS Injector plugin - https://github.com/n00bcodr/Jellyfin-JavaScript-Injector

2. Within Plugins > JS Injector, add a script, name it "Maintenence banner" or similar for reference, then paste the raw code of the jellyfin-maintenance-banner.js file in this project, then save.

3. Hard refresh your browser (ctrl + f5, shift + refresh, or clear site data) and test!
